### Usage

#### First Install

```bash
npm install react-native-modal-translucent --save
```

**or**

```bash
yarn add react-native-modal-translucent
```

#### Last Link

```javascript
$ react-native link react-native-modal-translucent
```

That is All !

Now go to see the effect.
